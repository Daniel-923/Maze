# Maze

해당 게임은 C++로 만든 3D 1인칭 미로게임입니다. 

미로는 Recursive Backtracking 알고리즘을 사용하여 맵을 생성했습니다. 

사용자는 앞, 뒤로 움직일 수 있으며 좌우로 방향을 조정할 수 있습니다.

미로 들어갔을 때부터 목적지에 도착할 때까지의 시간으로 순위를 결정합니다.
